Welcome to Asana

1 Download Asana from App Store or Google Play
2 Create account with email or Google/Apple sign-in
3 Complete initial setup: name, photo, and work preferences
4 Choose a template or start with a blank workspace

First Project Setup

1 Tap the + icon to create your first project
2 Name your project and choose privacy settings
3 Select layout: List, Board, Timeline, or Calendar view
4 Add team members by tapping "Share" button
Start with a simple personal project to learn the interface before managing team work.

Creating Tasks

1 Tap the + button or "Add task" in any project
2 Enter task name and press return for quick add
3 Tap task to add details: description, due date, assignee
4 Add subtasks for breaking down complex tasks

Task Management

1 Swipe right on task to mark complete
2 Swipe left to access more options (delete, duplicate)
3 Drag and drop tasks to reorder or move between sections
4 Use filters to view tasks by assignee, due date, or status
Use keyboard shortcut "Tab+Q" on desktop to quickly add tasks. On mobile, use voice input for faster task creation.
